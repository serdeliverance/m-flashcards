# m-flashcards

Flashcards for learning languages. This project born from a real need and also comes in handy to dive deeper into `Nextjs` and related technologies.

## Stack

- `Nextjs`
- `Typescript`
- `Mongo`
- `Docker`

## Run app locally

```
yarn dev
```

It runs the frontend on localhost:3000

## Start local DB

run `docker-compose`:

```
docker-compose up
```
